# 更新日志

本项目所有版本的变更都记录在此。版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.3.0] - 2026-06-28
### 新增
- **全局快捷键**：`Ctrl+Alt+1 / 2 / 3 …` 直接切换到对应屏，无需鼠标点横条，远程会话里更顺手。
  启动时按显示器数量自动注册，🔄 重新检测后同步更新。横条按钮上直接标注对应快捷键。

## [0.2.0] - 2026-06-27
### 新增
- **检查更新机制**：启动时以及点击 🔄 时，调用 GitHub API 查询最新 Release 版本。
  发现新版会在横条下方提示「🔔 发现新版 vX.Y.Z → 点击下载」，点击用系统浏览器打开
  Release 页面下载。网络不通时静默跳过，不影响离线使用。

## [0.1.1] - 2026-06-27
### 修复
- 切换屏幕后窗口未跟随、留在旧屏导致切不回去的问题。根因是 `ChangeDisplaySettingsEx`
  异步重排桌面坐标系产生的时序竞态：改用物理坐标（避免跨屏 DPI 缩放歧义），并在切换后
  约 750ms 内反复重置窗口位置，确保坐标系稳定后窗口落在新主屏。

## [0.1.0] - 2026-06-27
### 新增
- 首个版本：可视化横条切换主显示器，让 toDesk / AnyDesk / RustDesk / 向日葵等
  只串流一块屏的远程软件，能查看被控电脑的任意一块显示器。
- Win32 枚举显示器（`EnumDisplayDevices` / `EnumDisplaySettings`）。
- 通过 `ChangeDisplaySettingsEx` 切换主显示器；切换后窗口自动跟随到新主屏。
- 纯逻辑 `compute_layout` 单元测试覆盖坐标平移计算。
